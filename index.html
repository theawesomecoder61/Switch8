    Getting started

    On your mark, get set, go!

    Getting started is super easy!

    CDN (we host it for you)

    1. <link rel="stylesheet" href="http://theawesomecoder61.github.io/Switch8/switch8.css">
    2. <script src="http://theawesomecoder61.github.io/Switch8/switch8.js"></script>

    Note! The links above are hosted on this website, so they may be slow sometimes.

    Locally (it's on your computer)

    1. <link rel="stylesheet" href="path/to/switch8.css">
    2. <script src="path/to/switch8.js"></script>

    Using Switch8

    The instruction manual

    HTML

    1. <input type="checkbox" class="switch8">

    HTML customizations coming in version 2.5!

    or

    { } Javascript

    1. var beautifulSwitch = new Switch8({ elem: "#myCB" });

    You can customize your Switch8 in JS!

    Options

    Customize your Switch8

    elem - string

    Defines the element

    htmlAttr - boolean

    If it's true, you can use HTML attributes to customize the switch

    checked - boolean

    Sets the switch's state

    disabled - boolean

    Prevents user from toggling the switch, also looks ugly :)

    -- HTML customizations --

    data-checkedColor="" - string (hex, rgba)

    Changes the color of the checked state

    data-uncheckedColor="" - string (hex, rgba)

    Changes the color of the unchecked state

    -- Javascript customizations --

    checkedColor - string (hex, rgba)

    Changes the color of the checked state

    uncheckedColor - string (hex, rgba)

    Changes the color of the unchecked state
