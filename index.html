    Getting started

    Super easy!

    CDN

    <link rel="stylesheet" href="https://raw.githack.com/theawesomecoder61/Switch8/master/switch8.css">

    <script src="https://raw.githack.com/theawesomecoder61/Switch8/master/switch8.js"></script>

    Note! The links above are hosted on this website, so they may be slow sometimes.

    Locally

    <link rel="stylesheet" href="path/to/switch8.css">

    <script src="path/to/switch8.js"></script>

    Using Switch8

    HTML

    <input type="checkbox" class="switch8">

    HTML customizations coming in version 2.0!

    or

    Javascript

    var beautifulSwitch = new Switch8({ elem: "#myCB" });

    Options

    elem - string

    Defines the elemen

    checked - boolean

    Sets the switch's state

    disabled - boolean

    Prevents user from toggling the switch

    checkedColor - string (hex, rgba)

    Changes the color of the checked state

    uncheckedColor - string (hex, rgba)

    Changes the color of the unchecked state
